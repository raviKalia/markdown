## CoreScore – Cricket Scoring App

**CoreScore** is a cricket scoring web app created using HTML, CSS, and JavaScript.

### Features

- Tracks runs, wickets, and overs
- Supports two innings
- Shows last over and current over
- Includes undo functionality
- Stores match history using localStorage

**Tech Used:** HTML, CSS, JavaScript, MySQL
